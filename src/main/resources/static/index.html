FILEZ

Welcome to FileZ, this is project with the goal of self-hosted file server.
And also multi-user service. Where the users can share file between them.

Multiple Users

Create multiple accounts, upload and manage files, and share content securely between users.

Custom Limits

Set custom storage limits per account to ensure fair usage, prevent abuse, and manage server capacity efficiently.

Open Source

Built with transparency at its core, the entire platform is fully open source. You can inspect the code, customize every detail, and host it your own way without restrictions.
